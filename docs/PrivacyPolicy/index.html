プライバシーポリシー

このプライバシーポリシーは、kuroyoが提供するアロマサーチアプリが取得する個人情報の取り扱いを定めるものです。

本アプリは、本プライバシーポリシーの内容をご確認、ご同意の上、ご利用ください。


1.機能

　本アプリは、以下の機能を提供します。

　・プリセットされたエッセンシャルオイルの情報の表示・検索

　・プリセットされたクラフトレシピ情報の表示

　・ユーザー編集可能な調合情報の表示


2.本アプリが取得する情報、取得方法

　本アプリはApp Store Connectより以下の情報を取得いたします。

　・アクティブなデバイス数

　・クラッシュしたアプリ数


３.本アプリが取得する情報の利用目的

　本アプリは取得した情報により今後のアプリ改善に役立てます。
